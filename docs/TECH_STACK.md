# Tech Stack

This project was built with a modern web development stack designed for high-performance and premium design.

---

## Core Technologies
- **Framework**: [React](https://reactjs.org/) (Version 18.x)
- **Build Tool**: [Vite](https://vitejs.dev/) (Version 6.x)
- **Language**: [TypeScript](https://www.typescriptlang.org/) for type safety and scalability.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Version 3.4) with custom configurations for a premium "tech" aesthetic.
- **Asset Management**: Centralized cloud hosting via Google Drive, managed through `src/assets/assetLinks.ts`.

---

## UI Components & Design
- **Primitives**: [Radix UI](https://www.radix-ui.com/) for fully accessible, unstyled components (accordions, dialogues, dropdowns, etc.)
- **Icons**: [Lucide React](https://lucide.dev/) for consistent, customizable SVG icons.
- **Charts**: [Recharts](https://recharts.org/) for data visualization on the Skills and Projects pages.
- **Carousel**: [Embla Carousel](https://www.embla-carousel.com/) for smooth horizontal layout interactions.
- **State Management**: [TanStack Query](https://tanstack.com/query/latest) (React Query) for efficient data fetching and caching.

---

## Utilities
- **Type Safety**: [Zod](https://zod.dev/) for runtime schema validation.
- **Forms**: [React Hook Form](https://react-hook-form.com/) for lightweight and high-performance form state.
- **Date Utilities**: [date-fns](https://date-fns.org/) for modern date manipulations.
- **Animations**: Using a combination of custom Tailwind animations and native CSS transitions.

---

## Fonts
- **Inter**: (Google Font) Used for main UI and informational text.
- **JetBrains Mono**: (Google Font) Applied to code snippets and technical markers to emphasize the developer brand.
