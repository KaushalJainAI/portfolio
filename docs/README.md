## Project Purpose
This portfolio was built for **Kaushal** to showcase a blend of high-end software engineering skills and creative research expertise. It features a modern, tech-forward aesthetic with a strong emphasis on clean typography, responsive design, and unique 3D/tech-inspired imagery.

### Key Values
- **Premium Aesthetics**: Uses a sleek dark-mode-first approach with custom brand colors and smooth animations.
- **Performance**: Built on Vite and React for ultra-fast load times and developer productivity.
- **Responsiveness**: Fully optimized for mobile, tablet, and desktop views.
- **Accessibility**: Uses semantic HTML and Radix UI primitives for accessible UI components.
